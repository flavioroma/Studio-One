## Run Locally

**Prerequisites:**  Node.js
Get the latest LTS version of Node.js® for your Operating System (https://nodejs.org/en/download) and install it.

**Install**

1. Create a folder and clone the repository from the github page
2. From the new folder, install dependencies:
   `npm install`
3. Run the app:
   `npm run dev`

Use Chrome-based browsers for the best experience.
